WeatherFlow — Dashboard de Clima

Aplicação Web · Projeto Pessoal
Porto Alegre

Dashboard meteorológico interativo que consome a OpenWeatherMap API em tempo real. Exibe temperatura atual, sensação térmica, umidade, velocidade do vento e previsão para 7 dias com gráfico de barras animado. Inclui busca por cidade, geolocalização automática e troca de unidades (°C / °F). Layout totalmente responsivo com transições suaves.

TaskForge — Gerenciador Kanban

Aplicação Web · Projeto Pessoal
A Fazer
Em Progresso
Concluído

Aplicação Kanban de gerenciamento de tarefas com drag-and-drop nativo via HTML5 API. Permite criar, editar e mover cards entre colunas (A Fazer, Em Progresso, Concluído). Os dados são persistidos no localStorage, garantindo que as tarefas sobrevivam ao reload da página. Inclui filtros por prioridade, estimativa de tempo e tags coloridas.
